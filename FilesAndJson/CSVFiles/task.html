In this task you will have to complete two classes:

  1. The Task class
  2. The Database class.

Resources

If you look at the project, at the bottom there is a folder named resources. In this folder you will see a file called iris.data. This is a text file. Double click on it and look at its contents.

iris.data is what is known as a comma separated values file (or CSV file). That is, if you think of the file as a grid, each line is a row and commas delimit columns.

For example, the first three lines of the file are:
s-length,s-width,p-length,p-width,species
5.1,3.5,1.4,0.2,Iris-setosa
4.9,3.0,1.4,0.2,Iris-setosa

Which is equivalent to:
s-lengths-widthp-lengthp-widthspecies
5.13.51.40.2Iris-setosa
4.93.01.40.2Iris-setosa

The first line of iris.data contains the headers of the columns.

The following lines contain the actual data.

Classes (Your job)

  1. The purpose of the Database class is to stores the contents of a CSV file in several variables and to facilitate access to those contents. In the Database class you will have to complete two methdos
    • the Constructor, that takes in the contents of a CSV file as a String: In here you should put code to initialize all of the member variables of the class:
      1. colNames: is an array with the column names (from the first row)
      2. numRows: is the total number of data rows (not counting the first line with the column names)
      3. data: is a 2D array that contains the data as a grid of String objects.
    • The getValue method takes a column name and a row number and retrieves the corresponding element.
  2. The Task class has one method to compete: readCSV.
    • This method takes in a file name as parameter and returns a Database object
    • You have to complete the bits and pieces to take the file name and read the contents of the file into the String contents. Then use contents to initialize a Database object and return it.
    • A special note on reading byte[]s: You can convert a byte[] to a String with a special constructor for the String class. This constructor takes in the byte[], the starting position of that array, and the number of elements to read from it. Check the Javadocs for more information.

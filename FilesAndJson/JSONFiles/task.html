Another Database

The task for this module is to complete another two classes named Database and Task.

The idea is that you will have a Database of restaurants with minimal functionality.

Specific Tasks

  • Task has two methods besides main.
    • readFile: You should complete this method first. It is similar to the readFile method in the CSVFiles task, but it uses a BufferedReader. Your job is to complete it and return a String with the contents of the file.
    • readJson: Code this method second, as you can use the previous method for help. This method takes a filename (String) and reads the file as a String and creates a JSON document from it. Then, returns it.
      • Check the last slide from io to find out how to create a JsonObject from a String.
    • Database has two methods and a constructor.
      • The constructor should set a member variable. You pick the name.
      • getRestaurant. Given the name of a restaurant, this method returns the JSON object corresponding to that restaurant.
      • getAverageReviews: This method should lookup the reviews for a given restaurant and obtain the average. To obtain the total size of a JsonArray you can use the size() method.

      Don't forget to run the "check" until you get the task in the best shape you can

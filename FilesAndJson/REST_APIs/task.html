The Ultimate TV Show Database

The task for this module to complete the class TVShowsDatabase. This class connects to a website (TVMaze.com) and allows you to query shows by name or query celebrities that appear in shows. The website returns its results as a list of JSON documents. It is an API.

Specific Tasks

  • TVShowsDatabase has five methods and a default constructor.
    • You have to create a getURLSource method that takes in a URL. Based on the URL given, the method queries the online site and brings the contents of it as a String. You have to watch the videos to get this. This is not in the slides. Lastly, because this is a string that contains a list of JSON documents, you need to make it look like a proper JSON document with a root element. That is, append {'result': at the beginning of the string and } at the end. This effectively creates a JSON string with the root element "result" and the value will be a list; the list returned by the website.

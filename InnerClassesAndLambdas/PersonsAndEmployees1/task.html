Persons and Employees

Task 1. Simple Member Inner Classes

In this class you will build a Person class. Now, that person will be an employee of a company, with a position and a salary. However, if you have a class that represents a generic person, the position and salary can be conceptually separated from the basic data by using an inner class.

For this task you will have to:

  • Create a Person class in the file Person.java
  • Import the java.time.LocalDate and java.time.Period classes. You will need them later.
  • Create a two private member variables to hold the name and the birthDate of the employee. The birthdate has to be a LocalDate
  • Create a private member inner class called Position (non-static).
  • Position has two package-protected member variables. One to hold the name and another for the salary of the employee. The salary has to be a double
  • Position overrides the toString() method and prints:name+":"+salary.
  • You must also add a member variable in Person called position of type Position.

  • Create a constructor for Person that takes in two Strings. A name and a birthdate in ISO format (yyyy-mm-dd) and sets the corresponding memeber variables correctly. Key here is to go from a String to a LocalDate to be able to set the birthDate variable correctly.

  • Create a method called getAge() that returns an int with the age of this person in years. You have to use the Period class to compute the period between the date right now and the birthdate, and convert it in years. This is your research.
  • Create a method called setPosition that takes in a String and a double for the title of the position and the salary. This should create a Position object and assign it to the member variable position.
  • A getter for position
  • Override the toString() method to display the person's name, age and position (with salary and title).

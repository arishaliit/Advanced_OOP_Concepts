Resume Builder

Task 2. Static v. Non-Static Inner Classes

In this class we are going to expand on Task1. We will create the same Person class and we will add a new class Resume.

The specifications are as follows:

  • Copy and paste the Person class from task 1 into Person.java
  • Inside the Resume class:
    • Create a public member inner class called Education. This class will represent the education of a person.
    • Education must have two member variables to hold the name of the school and the major.
    • Education should have getters and setters for these variables, and a constructor that initializes the school and the major, in that order
    • Education must override the toString() method to print the major, the word at and the name of the school
    • Create a public and static inner class called Experience. This will represent the experiences of the individuals. For example: "VP of Sales" from 2018 to 2019
    • Experience must have a variable to hold the title of the job, and two integers to hold the start year and end year of that position.
    • Experience must have a constructor that initializes the title, the start date and the end date in that order.
    • Experience overrides the toString() method to print the start year, a dash, the end year and the title. For example: 2018-2019: VP of Sales
  • In addition, the Resume class should have:
    • A private member variable of type Person
    • Two private member variables of type List. One with objects of type Education and one with objects of type Experience
    • A public constructor that takes in a Person variable and sets the Person variable and makes all necessary initializations for the other variables.
    • A public void method addExperience that receives an object of type Experience and adds it to the corresponding List
    • A public void method addEducation that receives an object of type Education and adds it to the corresponding List
    • Override the toString() method to include the Person's name, age and actual position on the first line. Then a line that says "Experience", then all the Experience objects, then a line that says "Education" and below it all the Education objects. Thus, creating a resume.
  • Finish the TODOs in the Task.java file.

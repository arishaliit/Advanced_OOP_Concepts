Task 3. Anonymous Inner Classes

This is almost exactly as Task 2. A lot of the code can be copied and pasted in the corresponding files. In fact, I recommend you do that.

The Person class is exactly the same as in Task 2. It is safe to copy it exactly as is. The Task class is also exactly as in Task 2. Copy it into its file. Lastly, you can copy the code in the Resume class in pieces. There are very few differences with Task 2.

The differences you have to figure out and change are:

  • The List variable to hold the Experience objects will be implemented as a PriorityQueue. The idea is that no matter the order in which you enter the experiences, they will be read from most recent to least recent.
  • The constructor should initialize the priority queue with an anonymous Comparator class. The compare method should be implemented to return a negative value if the second's object end date is bigger than the first objects end date, and a positive value if the second date is smaller than the first. You must use the keyword new within the parameter of the constructor of the priority queue.
  • Lastly, add a standard getter for experience: getExperience()

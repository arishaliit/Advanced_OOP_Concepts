A Car Race.

This task will take you into creating Cars, Racers and then racing the cars.

Car

This class has four member variables:

  • miles:The number of miles left to travel with the current fuel
  • odometer: total miles travelled by the car
  • name: The name (brand) of the car
  • maxMiles: The maximum number of miles this car can travel when full.

Implement the move method. When this method is called, the miles left in the car diminish by the amount passed into the method. The odometer adds those miles to its count. If the miles in the car reach zero, the car does not move. That is, it does not discount more miles nor add to the odometer.

Racer

This makes a regular car into a racer. That is, a car that can race.... in its own thread

  • In the class you need to implement the Runnable interface, so that Racer objects can run in a thread
  • Create a constructor that takes in a Car object and initializes this Racer
  • Because this class is Runnable, you need to implement one method: run. The implementation of run should do the following: As long as the car's odometer has not reached the distance this racer is supposed to race, the car will move 1 mile. If the car runs out of fuel (i.e. 0 miles left in the tank), the car needs to charge before moving. When the car charges, it needs to print "Charging" and the current thread should sleep as many milliseconds as miles were charged. For example, in Task.java "Kona" will charge for 250 milliseconds, while "Ioniq" will charge for 220. This basically simulates a slower re-fueling speed for cars with a bigger range/tank capacity

A Car Race.

This task is a slight modification of the previous task. You will use Executors.

Car

This class is exactly the same as the Car in Task 1.

Racer

This has exactly the same implementation as the Racer in task 1.

Task

  • You need to declare a static member variable called service of type ExecutorService
  • In main, set racer1 and racer2 to be Racer objects with each Car object
  • You also need to create and implement the initExecutorService. This method should initialize service creating a new FixedThreadPool with a minimum of 3 threads.
  • In main, submit both Runnables to the service and then request a shutdown of the service.
    Requesting a shutdown will finish executing the threads in the pool and then stop the pool and finish the program.

An Webpage editor.

This task will have you create a very simple web page editor with an auto-save feature.

You type your webpage line by line. If a line begins with t: it indicates a title. Otherwise it is a normal line.

The webpage will aauto-save every 10 seconds.

WebPage

  • In the constructor, initialize the dom to an empty List as you see fit.
  • addTitle will add an element to the dom that consists of the parameter surrounded by <h1> and </h1>
  • Similarly, implement addParagraph which will do the same as title, but the tags are <p> and </p>
  • The toString method is missing the dom elements. Please add them to html
  • save will save the string representation of this webpage to the filename provided.<
    You already have a method that provides a String representation of this webpage. Also, using a FileWriter makes this a very short method.

WebPageEditor

  • Implement the constructor: Initialize wp to an empty webpage, and filename to the value passed.
  • In the start method:
    • Using Executors, instantiate a new SingleThreadedScheduledExecutor
    • Schedule at a fixed rate a Runnable that executes checkSave. if checkSave is true it will print "Saved", This has to be scheduled to start in 10 seconds and repeat every 10 seconds <
      Check the Executor api or the slides to see how you can schedule at a fixed rate. Also, the runnable can be implemented with the ()->{ } notation.
    • The next section creates a Scanner to read the user input line by line (the actual text editor). In the while loop you have to read a line, and If the user types "quit" the loop ends. If the user input starts with t: you add a title to the webpage. Otherwise, you add a paragraph.
    • Make sure that when the while loop ends, the service is shutdown NOW.
    • As an example, if you were to type: t:Title, and This is a java program in the second line, and then quit after 10 seconds, you should have a file with the contents.
      <html><head></head><body>
      <h1>Title</h1>
      <p>This is a java program</p>
      </body></html>
  • checkSave does the following: If the string representation of wp is equal to its last String representation (stored in tmp) the method returns false. Otherwise, it saves the string representation of this webpage to the given filename (using WebPage methods), and tmp updates to the string representation that you just saved. Then, the method returns true.

Task

  • You need to declare a static member variable called service of type ExecutorService
  • In main, set racer1 and racer2 to be Racer objects with each Car object
  • You also need to create and implement the initExecutorService. This method should initialize service creating a new FixedThreadPool with a minimum of 3 threads.
  • In main, submit both Runnables to the service and then request a shutdown of the service.
    Requesting a shutdown will finish executing the threads in the pool and then stop the pool and finish the program.
